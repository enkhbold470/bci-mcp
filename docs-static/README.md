# Static Documentation for BCI-MCP

This directory contains the static HTML files for the BCI-MCP documentation site deployed on GitHub Pages.

## Structure

- `index.html` - Main documentation page
- `styles.css` - CSS styles for the documentation
- `favicon.svg` - Favicon for the site

## Development

To make changes to the documentation:

1. Edit the HTML and CSS files directly
2. Commit and push changes to the main branch
3. GitHub Actions will automatically deploy the updated site

## Deployment

The documentation is automatically deployed to GitHub Pages using the workflow defined in `.github/workflows/deploy-docs.yml`.

No build step is needed as these are static HTML files.
